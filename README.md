 
# 📸 Minimal Photography Portfolio

 *Check it here:* https://coddiction-101.github.io/PhotographyPort/

A clean, high-contrast, and image-focused portfolio layout designed to let the photography speak for itself. Built with a focus on minimalism, typography, and smooth micro-interactions.

## 🚀 Features

- **Minimalist Design:** High-contrast layout with ample whitespace for a premium feel.
- **Sticky Navigation:** A modern navigation bar with a glassmorphism (backdrop-filter) effect.
- **Responsive Masonry Grid:** Auto-adjusting gallery that looks great on all screen sizes using CSS Grid.
- **Micro-interactions:** Smooth grayscale-to-color transitions and scale effects on image hover.
- **Semantic HTML:** Clean, accessible, and SEO-friendly code structure.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, Flexbox, and CSS Grid.
- **Google Fonts:** 'Inter' for modern typography.
 
## 🔮 Future Features to Add

To further improve this portfolio, here are the planned enhancements:

1.  **Lightbox Gallery:** Implement a "click-to-expand" feature using JavaScript to view images in full-screen mode.
2.  **Lazy Loading:** Add native `loading="lazy"` or a JS-based intersection observer for better performance when handling high-res images.
3.  **Dark/Light Mode:** A toggle to switch between a classic white gallery and a "darkroom" aesthetic.
4.  **Category Filtering:** Add buttons to filter images by category (e.g., Nature, Portraits, Street).
5.  **Dynamic Data:** Move the gallery data to a JSON file or a CMS to manage photos without editing HTML code.
6.  **Contact Form:** A functional contact section using Formspree or a backend service.
7.  **Page Transitions:** Smooth fade-ins when navigating between sections.

## 📝 How to Use

1.  Clone the repository.
2.  Replace the `src` in the `<img>` tags with your own photography links or local paths.
3.  Customize the `.logo` and `h1` text in `index.html`.
4.  Open `index.html` in any modern browser.
